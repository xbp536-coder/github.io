What is a CPU cache? (short explanation)

Date: 10 February 2026

A CPU cache is a small, very fast type of memory inside the processor. It stores copies of data that the CPU uses often, so the CPU does not have to wait for slower main memory (RAM). This makes programs run faster.

Intro to normalisation in databases

Date: 10 February 2026

Normalisation is the process of organising tables in a database so that each table focuses on one topic and there is minimal duplicated data. This helps keep the data consistent and easier to maintain.
